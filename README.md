# Rule-Based AI Chatbot

Basic Rule Based python based AI Chatbot.  
The chatbot waits for the user to enter a command and responds accordingly based on the user's command inputted using if-else statements, and can continue to work until the user exits the program.
This project is carried out as a part of an Artificial Intelligence Internship Project.

# Features:
- Responds to greetings
- Answers basic questions
- Handles exit commands
- Says YES or NO in response to rules
- An ongoing conversation between the chatbot and the user.
- Beginner-friendly AI project

# Technologies Used:
- Python
Visual Studio Code (VS Code)—A code editor.

# How to Run the Project:
Step 1: Install Python
   Download Python from the official website:
    - https://www.python.org
     Open Project Folder in VS Code.
     
Step 2: Open the project folder in VS Code.
    Open the folder which contains:
      - chatbot.py
         In Step 3, the Program is executed.
         
Step 3: Execute the Program.
  You can open terminal and type in:
    - python chatbot.py

  - If you don't have python installed:
    - python3 chatbot.py
